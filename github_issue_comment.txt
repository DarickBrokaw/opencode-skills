**Analysis:**
This is a Floating UI positioning library issue. The error occurs when:
1. A floating element (dropdown/tooltip/suggestion box) tries to calculate its position
2. The element hasn't been fully added to the DOM yet
3. `autoUpdate` calls `getComputedStyle` on an undefined/null element reference

**Triggers:**
- Long input strings/prompts causing layout recalculation
- May be related to "Animate Items" settings
- First message interaction often triggers it

**Workaround applied:**
Disabled auto-update in config to prevent version changes. Please re-open or address if this is still being tracked.
